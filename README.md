# Youtube Downloader


Aplicativo em Python e Flet para Download de vídeos e playlists do YouTube usando a biblioteca [PyTube](https://github.com/nficano/pytube)

## Instalação

```python
pip install -r requirements.txt
```

## Execução

```python
python main.py
```
